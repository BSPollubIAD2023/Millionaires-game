# Millionaires

A desktop application recreating the popular Polish TV game show "Millionaires". Test your general knowledge and compete with other players.

## Features
- Play a quiz inspired by the TV show "Millionaires".
- Save your score together with your username.
- Compare your results with other players.
- Add your own question sets.
- Edit questions directly in the application.
- Alternatively, modify the Pytania.txt file manually.
- Instructions for creating questions are displayed inside the application.
- Technologies
- Python
- PyQt5
